About CNA

联盟简介

中餐北美品牌联盟（CNA）于2025年11月13日在上海第28届FHC环球食品展期间正式成立，由大会组委会及11家企业共同发起，首批吸纳14家行业标杆作为理事单位。

2026年4月1日在第二届中餐全球化发展论坛期间扩大到20余家连锁品牌及核心企业。大家共同发起成立"全球连锁生态联盟（筹）"并一致决定先依托中餐北美品牌联盟开展工作。

Founding Context

发起背景

当前，中餐出海正迎来历史性发展机遇。数据显示，2024年中国餐饮连锁化率达24%，全球中式餐饮门店数量已超过70万家。

国家政策层面持续发力，2024年商务部等九部门联合发布《关于促进餐饮业高质量发展的指导意见》，明确提出"加快中餐走出去"的战略方针。

2025年国务院发布《关于深入实施"人工智能+"行动的意见》，为"人工智能+消费提质"与"人工智能+全球合作"注入新动能。

然而，中餐在出海过程中仍面临海外厨师匮乏、劳务输出政策限制、供应链食材不标准、跨境物流成本高等多重挑战。

中餐连锁"中国模式"与"中国智能"深度融合的优秀案例不断涌现，以连锁化、标准化、智能化为内核的中国特色餐饮全球化路径正在被验证、被复制、被输出。

中餐北美品牌联盟，以"模智双驱，抱团取暖"为战略主轴，全面推动中餐从"走出去"迈向"走进去、走上去"。
